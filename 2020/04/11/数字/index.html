1 python2和3的区别

python中有长整型，python3中没有

2 运算符

+ 加
- 减
* 乘
/ 除
// 取整
% 取余
** z指数，如3**2=9

当算数运算符后加上等于号，表示复合运算，如：

a += 1 等价于 a = a + 1

其他运算符都能使用这种操作。

3 强制类型转换

内置函数 说明 注意点
int 将数据强制转换成整数
long() 转成长整数类型 python2.7
float() 转成浮点数
complex(real[, imag]) complex(real[, imag]) -> complex number创建复数
str() 转换成字符串
eval() 将字符串中的内容，转成其本身的格式，如果是计算表达式，可计算出最终的结果 eval(‘1+1’)=2
repr() 将数据转换成字符串 eval(repr(obj)) == obj.
ord() 将一个字符转成 ascii码
chr() 将ascii码数字转换成字符串
hex() 转16进制数
oct() 转8进制数
